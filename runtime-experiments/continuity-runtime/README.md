## Purpose

This directory contains the first operational GitHub-native continuity runtime experiments.

The purpose is to validate:

- GitHub Pages runtime interaction
- workflow-generated runtime artifacts
- markdown/json continuity loading
- browser-extension-backed continuity augmentation
- repository-native continuity topology
